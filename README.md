# Creator Calc

Interactive revenue calculator for content creators. Model different pricing strategies, audience sizes, and income scenarios to find your path to sustainable income.

**[Live Demo →](https://lachlancb.me/tools/)**

## Features

- Slide-to-calculate pricing and audience inputs
- Multiple revenue stream modeling (sponsorships, products, services)
- Visual breakdowns of income scenarios
- Mobile-friendly dark UI

## Stack

- React + TypeScript
- Vite
- Radix UI components
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```
